# Backend Developer Portfolio - Dev Keshav Hire

A premium, minimal, and highly responsive personal portfolio website designed for a professional Backend Java Developer.

## Tech Stack
- **HTML5**: Semantic markup for accessibility and structure.
- **CSS3**: Vanilla CSS variables, transitions, and layout styles.
- **Bootstrap 5**: Grid layout, spacing classes, and responsive utilities.
- **Vanilla JavaScript**: Lightweight scroll logic, terminal interaction, and visual feedback.

## Directory Structure
```
portfolio/
├── index.html
├── css/
│     ├── style.css       # Global design variables and main layout styles
│     └── responsive.css  # Precise responsive breakpoint overrides
├── js/
│     └── script.js       # Navbar animations, ScrollSpy, and interactive terminal
├── assets/
│     ├── images/         # Image assets
│     ├── icons/          # SVG icons
│     └── resume/         # Resume download files
└── README.md
```

## Features
- **Backend Illustration**: Custom SVG illustration mapping system logic (Java, APIs, DB, connections) without distracting elements.
- **Interactive Terminal**: Code-based interactive CLI terminal demonstrating developer attributes.
- **Responsive Navigation**: Hamburger navigation on mobile devices with transparent-to-solid white transition.
- **Problem Solving Spotlight**: Highlighted visual elements calling out key problem-solving traits.
- **Accessibility & SEO**: Clean structural HTML, ARIA tags, and meta information optimized for web crawlers.
